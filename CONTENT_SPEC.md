# 천기 콘텐츠 기능 명세서

## 비로그인 맛보기 → 로그인 유도 전략
- 오늘의 운세/이달의 운세: 첫 2줄만 → "전체 보기 → 로그인"
- 오늘의 타로: 카드만 보임, 설명 잠금
- 행운 로또: 공통번호 → 로그인시 사주 맞춤번호
- 오늘의 명언: 전체 공개
- 꿈 해몽: 비로그인 1일1회 (IP), 로그인 1일3회 (계정)
- 띠별/별자리/혈액형: 결과 일부만 → 로그인 유도

## 로그인 필수
- 관상짤, 내 관상보기, 조선초상화
- YES/NO 타로, 궁합 연예인
- 모든 유료 콘텐츠
- 결과 저장/공유

## 궁합 연예인 (18번)
- 사주 오행 연예인 매칭 (무료)
- 기질도 연동: "기질도 먼저 하러 가기" 유도
- 오행 상생관계: 木→火, 火→土, 土→金, 金→水, 水→木
- 연예인 DB: 이름/생년월일/일간/오행/기질도유형

## 사주 월별운세 (20번)
- 입력: 이름+생년월일+이번달
- Claude API 동적 생성 (원가 ~50원)

## 신년운세 (22번)
- 입력: 이름+생년월일+올해연도
- 12달 + 항목별 (원가 ~100원)

## 토정비결 (23번)
- 입력: 이름+생년월일+태어난시간
- 만세력 괘 자동계산 → Claude 풀이

## 수비학 (24번)
- 소울넘버 1~9
- 피타고라스 수비학 기반
- 생년월일 숫자 더해서 계산

## 손금 (25번)
- 왼손 사진 → Canvas 선 오버레이
- 생명선/감정선/두뇌선/운명선 색깔별

## 발금 (26번)
- 왼발 사진 → 손금과 동일 방식

## 얼굴 점 (27번)
- 점 위치 감지 → "이 위치 점을 가진 사람의 특징"
- 연예인 매칭 ❌ (저작권)

## 타로 (9,14,29~33번)
- Rider-Waite 퍼블릭도메인
- 카드 뽑기 애니메이션
- 켈틱크로스(10장) 포지션 의미

## 12수호신 (35번)
- 무료, 12종 카드 슬라이드
- 굿즈 연결

## 기질도 (36번)
- 16문항 테스트
- 오행+음양 16가지 유형
- 캐릭터 이미지

## 이름 풀이 (38번)
- 한자별 뜻+획수
- 원형이정 분석
- 이름 오행+발음 오행
- 이름 총평 200자+
