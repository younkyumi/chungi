📜 오늘의 명언 — 풀 결과 점검본

📌 무료 콘텐츠 · "오늘의 명언" 전수 점검본. 직접 읽으며 오타·어색한 문구·중복을 잡아주세요.
• 가짓수의 정체: 책을 고르면, 그 책의 명언 배열에서 날짜 시드(seededRng) 기반으로 매일 한 문장이 자동 선택돼 결과 카드에 한 줄 떠요. (_getQuote() → seed = 연*10000+월*100+일 + bookId.length*77) 그래서 가짓수 = 전체 책의 모든 명언 문장. 아래에 책별로 전부 나열했어요.
• 총 명언 문장 수: 75개 (실제 도서 10권). + 협찬(AD) 도서 2칸은 quotes 비어있음(명언 0개) → 클릭 시 협업문의 이메일.
• 로그인 분기: locked:true 책(천기 제외 9권 전부)은 비로그인 시 잠금🔒 — "로그인하면 모든 책의 명언을 볼 수 있어요!" 알럿. 천기의 말씀만 locked:false(=기본/비로그인 열람 가능).
• 날짜 분기: 같은 책이라도 날짜가 바뀌면 다른 명언이 떠요(자정 갱신). 인물 선택은 없음(NO_PERSON_TODAY=["quote"]).
• 결과 카드의 정적 텍스트(헤더·해시태그·푸터 등)는 모든 책 공통 → 맨 위 "결과 레이아웃" 섹션에 1번만 정리했어요.
