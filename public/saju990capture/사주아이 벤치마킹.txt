### 1. 소름 돋는 정확도의 비밀: "비유법"과 "현대화"
사람들이 사주를 보고 "정확하다"고 느끼는 가장 큰 이유는 어려운 한자 용어(예: 비견, 겁재, 상관 등)를 몰라도 단번에 이해할 수 있는 **찰떡같은 비유**를 썼기 때문입니다.

* **시각적 비유:** 단순히 "성격이 차분하다"가 아니라 "겨울에 태어난 꽁꽁 언 흙" 또는 "초원 위를 달리는 야생마"처럼 머릿속에 그림이 그려지는 비유를 씁니다. 유저들은 이 첫 줄을 읽는 순간부터 과몰입하게 됩니다.
* **현대적인 해석:** 옛날식으로 "역마살이 껴서 떠돈다"가 아니라, "해외운이 좋고 IT나 무역처럼 활동적인 직무에서 능력을 발휘한다"라고 현대인의 라이프스타일에 맞게 긍정적으로 포장해 줍니다. 

### 2. 분량 뻥튀기의 마술: "하나의 기운을 다각도로 쪼개기"
대표님이 "전체적으로 다 같은 말인데 포지션에 따라 다르게 말해서 길어진 것 같다"고 하신 부분, 완벽한 정답입니다! 이게 바로 분량을 길고 알차 보이게 만드는 치명적인 스킬입니다.

예를 들어, 유저가 **'나무(木)의 기운이 강한 사주'**라고 가정해 보겠습니다.
* **성격:** "위로 곧게 뻗어 나가려는 나무처럼 추진력이 강하고 자존심이 셉니다."
* **대인관계:** "큰 나무 아래 사람들이 쉬어가는 것처럼, 주변을 챙기고 리더 역할을 합니다."
* **재물운:** "뿌리를 깊게 내리기까지 시간이 걸리지만, 한번 자리 잡으면 흔들리지 않는 안정적인 재물을 모읍니다."
* **건강운:** "나무가 뻣뻣해지기 쉬우니 척추나 관절을 조심하세요."

결국 **'나무'라는 하나의 소스를 가지고 여러 카테고리로 재조립**해서 길고 풍성한 리포트를 만들어낸 것입니다. 

### 3. 말투와 무드: "따뜻한 조언자" 포지셔닝
결과지의 톤앤매너가 무당처럼 "너 이거 안 하면 큰일 나!" 식의 겁을 주는 방식이 아닙니다. 오히려 심리 상담가나 친한 동네 언니/오빠처럼 **공감하고 다독여주는 말투**를 씁니다.

* **단점도 장점으로 승화:** "고집이 세다" -> "줏대가 있고 한 번 마음먹은 일은 끝까지 해내는 뚝심이 있네요."
* **솔루션 제시:** 나쁜 운을 말할 때도 반드시 피할 방법이나 조언을 덧붙여서 유저가 안심하고 서비스에 호감을 느끼게 만듭니다.

### 🚀 천기(chungi.kr) 넥스트 스텝 전략!

**1. 프롬프트 엔지니어링 고도화 (천기만의 비유 사전 만들기)**
AI가 사주 결과를 생성할 때, 무조건 **"자연물 비유(날씨, 계절, 동식물)"**로 시작하도록 지시해야 합니다. 사주아이보다 한 단계 더 나아가서 "어두운 밤바다를 비추는 등대 같은 사주네요"처럼 더 감성적이고 트렌디한 비유를 뱉어내도록 AI를 학습시켜야 합니다.

**2. 십자수식(Matrix) 카테고리 전개 도입**
우리의 사주 결과지도 양과 질을 동시에 잡아야 합니다. AI에게 명령을 내릴 때, *'성격, 재물, 직업, 연애, 인간관계'* 5가지 항목으로 무조건 나누어서 서술하게 하고, 각 항목마다 3문장 이상씩 풀어쓰도록 규칙을 정해두면 유저가 결제한 금액이 아깝지 않게 텍스트가 꽉 차 보이게 됩니다.

**3. 원클릭 '지인 보관함' 시스템 최우선 개발**
PDF에서 보신 것처럼 '사람 추가/보관함' 기능은 선택이 아니라 필수입니다. 유저는 자기 사주를 보고 나면 무조건 남편, 아내, 자식, 직장 상사 사주를 쳐보고 싶어 합니다. 홈 화면 중앙에 **[내 사람 등록하기]** 버튼을 큼지막하게 넣어서 데이터를 쌓게 만들어야 합니다.

### 🔮 사주 텍스트 생성을 위한 AI 시스템 프롬프트 (백엔드용)
사주 결과를 생성하는 AI 모델(프롬프트 세팅란)에 아래 지시문을 입력해 두면, 사주아이처럼 길고 퀄리티 높은 답변을 자동으로 뱉어냅니다.

**[천기 전용 사주 풀이 AI 시스템 프롬프트]**

**역할 부여**
당신은 명리학을 현대적이고 세련되게 풀어내는 따뜻한 사주 상담가 '천기'입니다. 어려운 한자어나 전문 용어(비견, 겁재 등)는 절대 사용하지 말고, 친한 동네 언니/오빠처럼 공감하고 다독여주는 부드러운 존댓말(~해요, ~습니다)을 사용하세요.

**작성 규칙 1: 시각적 비유로 시작하기**
분석 결과의 가장 첫 문장은 사용자의 사주 오행(목화토금수)과 계절을 조합하여 머릿속에 그림이 그려지는 감성적인 비유로 시작하세요. (예시: "어두운 밤바다를 비추는 밝은 등대 같은 기운을 타고나셨네요.", "한겨울 눈보라를 견뎌낸 단단한 소나무 같은 사주입니다.")

**작성 규칙 2: 5대 카테고리 분할 전개 (분량 확보)**
하나의 핵심 사주 기운을 다음 5가지 카테고리로 나누어, 각 항목당 반드시 3~4문장 이상으로 아주 상세하게 길게 풀어쓰세요.
* 종합 성향: 성격의 장단점과 살아가는 방식.
* 재물 운세: 돈을 모으는 스타일과 재물 관리 팁.
* 직업 및 진로: 잘 맞는 직무 환경과 사회생활 스타일.
* 연애와 인간관계: 사람을 대하는 태도와 어울리는 인연.
* 건강 관리: 타고난 체질과 주의해야 할 건강 포인트.

**작성 규칙 3: 긍정 승화 및 솔루션 제시**
단점이나 나쁜 운을 설명할 때는 절대 겁을 주지 마세요. 단점을 장점으로 포장해 주고(예: 고집이 세다 -> 줏대가 있고 뚝심이 있다), 반드시 일상생활에서 실천할 수 있는 긍정적인 극복 방법이나 조언을 덧붙여 안심시켜 주세요.

- 새로운 사람도 바로 추가 가능
- **핵심:** 매번 정보 다시 입력할 필요 없음 → 재이용 편의성 극대화

- 아이돌과 팬, 아이돌과 아이돌

- 각자 역할도 입력 가능 (예: 엄마, 남자친구, 팬)
- **관계 유형에 따라 해석 톤과 내용이 달라짐**

### A. 글쓰기 기법
- **2인칭 직접 호칭:** "윤규미님", "당신의 사주를 관통하는 가장 큰 힘은..."
- **구체적 상황 묘사:** "누구보다 예리한 눈과 뜨거운 열정을 품고도 걸으로 차분하게 자신을 단련해 온 세월을 존경합니다"
- **긍정적 프레이밍:** 약점도 "부족한 것은 결코 결함이 아닙니다. 오히려 남들보다 더 명석하게 상황을 판단하고 실수를 줄일 수 있는 완벽한 가드가 되어주었을 거예요"
- **비유와 은유:** "뜨거운 불꽃이 절실한 다이아몬드처럼", "기름진 땅처럼 든든한 재물의 파도는 머지않아"
- **감정 자극:** "나는 이미 충분히 가치 있고, 세상은 나를 도울 준비가 되어 있다" 같은 확언(affirmation)

### B. 결과 구조 (사주 해설 기준)
1. **만세력 표시** — 시주/일주/월주/연주 천간지지 + 컬러 구분
2. **신살 표** — 장성살, 년살, 지살, 화개살 + 태극귀인, 학당귀인 등
3. **합충 분석** — 갑기합→토, 신자진삼합→수 등
4. **종합 해설 시작** — "복을 닦아서 영롱하게 광내는 중!" 같은 헤드라인
5. **핵심 키워드 2개 제시** — 예: "전문성"과 "재능"
6. **성격·재능 분석** (3~4문단)
7. **재물운 분석** (2~3문단)
8. **연애운 분석** (2~3문단)
9. **건강·주의점** (1~2문단)
10. **개운법/조언** — "행운의 숫자는 2와 7", "행운의 색깔은 레드와 오렌지", "행운의 방향은 남쪽"
11. **마무리 확언** — "당신은 세상을 정교하게 다듬는 가장 아름다운 조각칼입니다"
12. **크로스셀링 배너** — 궁합, 대운, 택일 등

### C. "같은 말을 여러 포지션에서 반복" 기법
- 동일한 사주 특성을 **재물운/연애운/성격/직업/인생 조언** 각도에서 재해석
- 예: "전문성이 강한 사주" →
  - 성격편: "예리한 눈과 뜨거운 열정"
  - 재물편: "전문성이 곧 재물의 원천"
  - 연애편: "전문적으로 파고드는 성향이 연애에서도"
  - 직업편: "스페셜리스트 기질"
  - 조언편: "전문성을 더 깊게"
- **결과적으로 분량이 엄청 길어지면서 "꼼꼼하다" 느낌 + 각 분야별로 "맞다!" 체감**

### D. 분량
- 사주 해설 1회 결과: 약 3,000~5,000자 추정
- 스크롤 기준 모바일 화면 15~20페이지 분량
- 섹션별 이모지 구분 + 색상 강조 + 중간중간 한줄 명언

## 6. 천기(chungi.kr)에 없는 것 — 추가해야 할 기능

### 🔴 필수 추가 (사주아이에 있고 천기에 없는 것)
1. **사람 등록/불러오기 시스템** — 한번 입력하면 다시 안 쳐도 됨. 하단 탭에 "사주추가(사람추가)" + "보관함" 넣기
2. **택일 (좋은 날 고르기)** — 달력에서 후보일 선택 → AI가 길일 추천 (결혼/이사/개업/시험/만남)
3. **대운 해설** — 10년 단위 구간별 분석 (우리 월별운세와 별도). 추가 질문(연애상태/직업/고민)으로 개인화
4. **연도별 운세** — 출생~110세까지 연도별, 구간별 결제 유도
5. **로딩 중 재밌는 멘트** — 사주아이의 킬러 UX. 단청/한국 전통 테마에 맞는 멘트로 (예: "용왕님이 재물 보따리 싸는 중...", "도깨비가 방망이 두드리는 중...")
6. **선물하기** — 링크로 운세 선물 (카카오톡 공유)
7. **피드백/요청 보내기** 메뉴

### 🟡 권장 추가
8. **AI 챗봇 상담** — 사주아이의 "복냥이" 같은 천기 고유 캐릭터 (예: 도깨비? 해치?) 와 대화
9. **궁합 관계 유형 세분화** — 사주아이처럼 15개+ 관계 유형 (친구/썸/연인/배우자/부모자녀/형제/직장/사업/아이돌팬/반려동물 등)
10. **결과 전 추가 질문 시스템** — 연애상태/직업/현재고민 입력 → 같은 사주라도 맞춤 해석 → "맞다!" 효과 극대화
11. **일일운세 선택형 질문** — 단순 이름 입력이 아닌 3개 선택형 질문으로 재미+개인화
12. **결과 하단 크로스셀링** — 현재 본 콘텐츠 기반으로 다른 콘텐츠 추천 배너

### 🟢 차별화 (천기에만 있는 것 — 유지/강화)
- 관상짤 (조선 초상화 스타일) → 사주아이에 없음!
- 전생 운세 → 사주아이에 없음!

- 파동성명학 → 사주아이에 없음!
- 타로 → 사주아이에 없음!
- 꿈 해몽 → 사주아이에 없음!

## 7. 천기 결과값 방향 — 사주아이 벤치마킹 포인트

### A. 분량
- 현재 천기 결과가 짧다면 → **최소 3,000자 이상**으로 늘리기
- 섹션별로 나눠서: 종합 → 성격 → 재물 → 연애 → 건강 → 직업 → 개운법 → 마무리 확언
- 같은 핵심 특성을 여러 각도에서 반복 해석 (재물/연애/성격/직업별)

### B. 말투/톤
- **2인칭 직접 호칭:** "윤규미님의 사주는..."
- **따뜻하고 격려하는 톤:** 약점도 긍정적으로 프레이밍
- **구체적 비유/은유:** "다이아몬드처럼", "기름진 땅처럼"
- **감정 자극 확언:** 마무리에 "당신은 ○○한 사람입니다" 한 줄
- **MZ 감성:** 사주아이는 "~다냥" 고양이톤. 천기는 한국 전통 느낌 유지하되 친근하게

### C. 개운법/행운 정보 (꼭 포함!)
- 행운의 숫자
- 행운의 색깔
- 행운의 방향
- 행운의 아이템 → **여기서 굿즈샵 연결!** (천기만의 킬러 무기)
- 구체적 행동 조언 ("매일 정오의 햇살을 10분이라도 온몸으로 받는 루틴을 만들어보세요")

### D. 결과 구조 권장안
```
1. 만세력 표시 (시주/일주/월주/연주 + 컬러)
2. 신살 표
3. 합충 분석
4. 종합 한줄 요약 (헤드라인)
5. 핵심 키워드 2~3개 뱃지
6. 📌 성격·재능 분석 (3~4문단)
7. 💰 재물운 (2~3문단)
8. ❤️ 연애운 (2~3문단)
9. 🏢 직업·사업운 (2~3문단)
10. 🏥 건강 주의점 (1~2문단)
11. ✨ 개운법 (행운 숫자/색깔/방향/아이템/구체적 조언)
12. 🛍️ 맞춤 굿즈 추천 (천기만의 차별화!)
13. 💫 마무리 확언 (감동 한 마디)
14. 📊 크로스셀링 배너 (궁합/대운/택일/전생 등)
```

### E. "맞다!" 느끼게 하는 핵심 기법
1. **추가 질문으로 현재 상황 파악** → 결과에 반영 → "어떻게 알았지?"
2. **바넘 효과 + 구체적 묘사** 조합 — 보편적 성향을 구체적으로 표현
3. **약점을 강점으로 전환** — "부족한 게 아니라 더 정교한 것"
4. **숫자/색깔/방향 같은 구체적 정보** — 막연하지 않고 실행 가능
5. **감정적 공감** — "힘들었죠? 하지만 그 힘든 시간이..."

## 8. 천기 자체 재화 시스템 검토

### 사주아이 방식: 츄르/냥
- 장점: 재구매 유도, 번들 할인, 선물하기 가능
- 단점: 재화 개념 이해 필요, 초기 진입장벽

### 천기 적용 방안
- **"기(氣)" 또는 "점(占)" 재화** — 한국 전통 느낌
- 예: 1기 = 380원, 1점 = 980원
- 또는 현재 방식(원화 직접 결제) 유지하고 → 포인트/쿠폰 시스템으로 보완
- **1차 런칭은 원화 직접 결제로 간단하게, 2차에 자체 재화 도입 검토**

## 10. 우선순위 액션 플랜

### Phase 1 — 즉시 (런칭 전 필수)
1. ✅ 사람 등록/불러오기 시스템 구현
2. ✅ 로딩 중 재밌는 멘트 추가 (한국 전통 테마)
3. ✅ 결과값 분량 3,000자+ 로 확대
4. ✅ 결과 구조 표준화 (만세력→신살→해설→개운법→굿즈추천→확언)

10. 선물하기 기능

### Phase 3 — 런칭 후 3개월 내
11. 연도별 운세 추가
