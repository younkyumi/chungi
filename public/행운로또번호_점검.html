🎰 행운 로또번호 결과 점검본

📌 점검 목적 — 앱의 행운 로또번호(무료) 결과 화면에 나오는 모든 정적 텍스트 가짓수를 한 화면에 펼쳤어. 직접 읽고 오타·어색한 문구를 잡아줘.

가짓수를 가르는 분기 변수는 단 2개:
① 로그인 여부 (isLoggedIn) — 비로그인 = 오늘의 공통 번호(오행 토 고정) / 로그인 = 사주 맞춤 번호(오행 근거·번호범위·기운설명 추가).
② 사주 일간 오행 (dominant) — 로그인 시 생년월일로 _getDominantOhang 계산 → 목·화·토·금·수 5가지. 이 값이 헤더 "기운" 문구, "오행 기반 번호 근거" 설명문, "주(主)" 표시 행, 해시태그 #{오행}기운을 바꿔.

⚠️ 번호 6개 자체는 날짜 시드 랜덤(_seededRng + 매일 자정 갱신)이라 텍스트 점검 대상이 아니야. 아래 카드의 공 번호는 예시일 뿐, 실제 당첨/문구와 무관해. 파란 글씨=분기로 바뀌는 부분, 골드 글씨=오행별 고유 설명.

따라서 대표 결과 카드 = 비로그인 1종 + 로그인 5오행 = 총 6종. 그 아래에 정적 텍스트 풀(인트로 5항목·분석중 4문구·오행 설명 5·번호범위 5·구매시간 고정값·면책)을 전수로 첨부했어.
