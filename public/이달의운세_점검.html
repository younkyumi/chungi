📅 이달의 운세 (무료 · monthly_unse) 전체 가짓수 점검본

📌 목적 — 천기 무료 "이달의 운세" 결과팝업에 들어가는 모든 텍스트 가짓수를 한 화면에 펼쳐 오타·어색한 문구·버그를 잡기 위한 정적 점검본이에요. 코드(src/app/page.tsx 21319~21815, MonthlyUnseModal)에서 텍스트를 그대로 옮겼어요.
파란 글씨/표 = 분기(가짓수) 변수 — 사주 일간 오행·월·점수구간·시드에 따라 달라지는 부분. 골드 = 고정 문구.
⚠️ 실제 화면은 시드(날짜+이름)로 각 배열에서 1개씩 뽑아 조합돼요. 여기서는 점검을 위해 배열 전체를 나열했어요. 한 사용자가 한 번에 보는 건 각 표에서 1행씩이에요.
⚠️ 주의 — 이건 점검용 HTML이라 실제 앱 스타일(화이트카드·골드)과 1:1 동일하지 않아요. 텍스트 내용 점검용이에요.
🔢 가짓수 구조 요약
① 이달 헤드라인+종합점수 : 5종 (★2~5) → 점수 = (별/5)×100
② 영역별 운 : 5영역(총운·재물·애정·건강·직업) × 각 4단계 = 20문구
③ 이달 절기 : 날짜로 24절기 중 1개 자동 매칭
④ 키워드 TOP 3 : 22개 키워드 풀에서 시드로 3개
⑤ 주차별 흐름 : 일간 오행 5종(목·화·토·금·수) × 4주차 = 20문구 (로그인 시에만 노출, 비로그인은 🔒 잠금)
⑥ 운명의 날 : 이름+연월 시드로 좋은날 4 / 주의날 2 자동 계산 (문구는 고정)
⑦ 1~12월 한 해 흐름 : 길[1,2,4,5,6,9,10,12] / 주의[3,7,11] / 보통 — 고정 (로그인+토정비결 매칭 시에만 노출)
⑧ Do & Don't : Do 20개 / Don't 20개 풀에서 각 1개
⑨ 천기의 한마디 : 14개 풀에서 1개
비로그인 vs 로그인 : 비로그인은 일간 "토" 고정·주차별 흐름 잠금·토정비결/1~12월 미노출. 로그인은 생년월일로 일간 오행 산출.

① 이달 헤드라인 + 종합점수 (5종)

시드(연·월·로그인여부)로 1개 선택. 종합점수 = round((star/5)*100). 별점은 ★ n개 + ☆ (5-n)개로 표시.
