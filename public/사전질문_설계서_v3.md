## ⚙️ 공통 UI 규칙

1. **단일 선택 Q** → 사실 정보 (연애 상태, 나이대, 현재 상황 등) — 하나만 선택
2. **복수 선택 Q** → "가장 궁금한 건?" 류 — 체크박스, 여러 개 선택 가능
3. **"전체 다 궁금해요!" 선택 시** → 모든 항목 다 포함해서 분석
4. **💭 기타 선택 시** → 텍스트 입력창 즉시 노출
5. **선택 후 자동 다음 단계** (기타·복수선택 Q는 확인 버튼 필요)
6. **건너뛰기 제공 기준**
   - ✅ 자유입력 포함된 Q
   - ✅ 다른 사람 대신 볼 때 모를 수 있는 정보 (Q3 자유서술 등)
   - ❌ 꼭 알아야 결과가 의미 있는 Q (연애 상태, 나이대 등)
4. **"강조 포인트" 개념 적용 콘텐츠** (전체 결과 기본 제공, 질문은 집중 영역 선택용)
   - 수비학 → 생년월일로 전체 결과 제공, Q는 강조 영역
   - 이름풀이 → 이름으로 전체 결과 제공, Q는 강조 영역
   - 손금·발금·얼굴점·눈점 → 사진으로 전체 보임, Q는 집중 부위
   - 관상보기 → 사진으로 전체 보임, Q1은 강조 영역
   - 토정비결 → 점괘가 핵심, Q는 집중 영역 설정용
   - 전생운세 → 사주로 전체 뽑아줌, Q2는 현재와 맥락 연결용
   → 건너뛰기 선택 시 모든 항목 균등 분석
6. **선택한 답변 → AI 프롬프트에 직접 반영**
   - "요즘 {Q1} 상황이시군요. {Q2}이(가) 가장 궁금하시죠?"
7. **📜 관운·문서운·시험운** → 사주풀이·진로·신년운세 등 전반 반영
